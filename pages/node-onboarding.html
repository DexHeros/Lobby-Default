Network Reachability

DexHero hosts relay traffic for each other — it's how the network avoids any centralized TURN server. Your node must be reachable on UDP for the matchmaker to trust it. This is mandatory.

Waiting for the DexHero Host agent to report its probe result…
Probe failed

Try these fixes in order:

  1. Forward UDP port 47989 in your router settings.
  2. Enable UPnP on your router (auto-opens ports).
  3. Enable IPv6 on your connection — no NAT required.
  4. Ensure you're not behind carrier-grade NAT (CGNAT) — some mobile / budget ISPs use it.
