Game Developer API

Integrate DexHero characters directly into your game. With your API key, you can fetch full character data — name, symbol, image, and 3D GLB model file — for every DexHero in the ecosystem.

Base URL http://localhost:8000/api/game
Who is this for? Game developers who have registered their game on DexHero and want to feature DexHero characters in their game world.

Authentication

All API requests require your API key passed as a header. Keys are tied to your registered game.

// Include this header in every request
X-API-Key: dh_live_YOUR_KEY_HERE
Never expose your API key in client-side code or public repositories. Always call the API from a server or backend.

GET /api/game/dexheros

Returns a paginated list of all DexHero characters with full character data including 3D model URLs.

Query Parameters

GET /api/game/dexheros

chain — Filter by chain: ethereum, bnb, base

limit — Results per page (max 100, default 50)

offset — Pagination offset (default 0)

// Example request
GET /api/game/dexheros?chain=ethereum&limit=20&offset=0
X-API-Key: dh_live_...

GET /api/game/dexheros/:id/owners

Returns all wallets that have permanently purchased/unlocked the DexHero. These players own the character outright with no expiry.

GET /api/game/dexheros/:id/access

Returns all wallets with active access — DexHero owners and staked players — merged and deduplicated. This is the primary endpoint for gating game login.

This is your game-login gate. Check the player's wallet against the access array at login. "owner" = permanent; "nft_holder" = active Play Pass holder.

JavaScript Example

// Fetch all DexHero characters from your backend
const res = await fetch('/api/game/dexheros?limit=50', {
  headers: { 'X-API-Key': process.env.DEXHERO_API_KEY }
});
const { dexheros } = await res.json();

dexheros.forEach(hero => {
  console.log(hero.name, hero.model_url);
  // Load hero.model_url as a GLB in your 3D engine
});

Unity (C#) Example

using UnityEngine;
using UnityEngine.Networking;
using System.Collections;

public class DexHeroLoader : MonoBehaviour {
    private const string API_URL = "https://yourserver.com/api/game/dexheros";
    private const string API_KEY  = "dh_live_YOUR_KEY"; // Store in secure config

    IEnumerator FetchDexHeros() {
        var req = UnityWebRequest.Get(API_URL);
        req.SetRequestHeader("X-API-Key", API_KEY);
        yield return req.SendWebRequest();

        if (req.result == UnityWebRequest.Result.Success) {
            Debug.Log(req.downloadHandler.text);
            // Parse JSON and load GLB model_url via GLTFast or similar
        }
    }

    void Start() => StartCoroutine(FetchDexHeros());
}
For loading GLB models in Unity, use GLTFast or Piglet. Pass the model_url string directly to the loader.

Unreal Engine (C++) Example

// DexHeroFetcher.h
#include "HttpModule.h"
#include "Interfaces/IHttpRequest.h"
#include "Interfaces/IHttpResponse.h"

void FetchDexHeros() {
    FHttpModule& Http = FHttpModule::Get();
    TSharedRef<IHttpRequest, ESPMode::ThreadSafe> Request = Http.CreateRequest();

    Request->SetURL(TEXT("https://yourserver.com/api/game/dexheros"));
    Request->SetVerb(TEXT("GET"));
    Request->SetHeader(TEXT("X-API-Key"), TEXT("dh_live_YOUR_KEY"));
    Request->SetHeader(TEXT("Content-Type"), TEXT("application/json"));

    Request->OnProcessRequestComplete().BindLambda(
        [](FHttpRequestPtr Req, FHttpResponsePtr Res, bool bSuccess) {
            if (bSuccess && Res.IsValid()) {
                FString JsonString = Res->GetContentAsString();
                // Parse with FJsonSerializer and load GLB via glTFRuntime plugin
                UE_LOG(LogTemp, Log, TEXT("DexHeros: %s"), *JsonString);
            }
        }
    );

    Request->ProcessRequest();
}

//  Access check: verify a player wallet before granting in-game access 
void CheckWalletAccess(FString DexHeroId, FString PlayerWallet) {
    FHttpModule& Http = FHttpModule::Get();
    TSharedRef<IHttpRequest, ESPMode::ThreadSafe> Request = Http.CreateRequest();

    FString URL = FString::Printf(
        TEXT("https://yourserver.com/api/game/dexheros/%s/access"),
        *DexHeroId
    );
    Request->SetURL(URL);
    Request->SetVerb(TEXT("GET"));
    Request->SetHeader(TEXT("X-API-Key"), TEXT("dh_live_YOUR_KEY"));

    Request->OnProcessRequestComplete().BindLambda(
        [PlayerWallet](FHttpRequestPtr Req, FHttpResponsePtr Res, bool bSuccess) {
            if (bSuccess && Res.IsValid()) {
                // Parse "access" array and check if PlayerWallet is present
                FString Body = Res->GetContentAsString();
                bool bHasAccess = Body.Contains(PlayerWallet.ToLower());
                UE_LOG(LogTemp, Log, TEXT("Access granted: %s"),
                    bHasAccess ? TEXT("YES") : TEXT("NO"));
            }
        }
    );

    Request->ProcessRequest();
}
To load GLB models in Unreal Engine, use the glTFRuntime plugin (free on GitHub) or Datasmith. Pass the model_url from the API response directly to LoadAssetFromFilePath or stream it via LoadAssetFromURL. The Epic skeleton from your DexHero will auto-bind to UE5's Mannequin animations.
